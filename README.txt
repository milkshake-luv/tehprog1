Это библиотека сделана для языка програмирования Python
и воложена в систему управления пакетами pip
Для её установки надо проделать стандартную процедуру установки
любой библиотеку прописать pip install fin_same
после чего пройдёт установка пакета
в программе производите стандартный import библиотеки
функция count_dif выполняет задание данной лабы, ей на вход даёте массив
она выдаёт количество неповторяющихся элементов на выходе
